01 // CLI INSTALLATION & QUICKSTART

Get Started with Mtrini CLI

1

Install from PyPI

Install the official PyPI package globally or in your Python environment.

pip install --upgrade mtrinicli
2

Launch Terminal Assistant

Type the main command in your terminal. Inside, type /ui for the TUI!

mtrinicli
⧉ Troubleshooting: If `mtrinicli` Command Is Not Recognized

If your terminal throws "mtrinicli is not recognized as an internal command" or "command not found", your Python Scripts folder is not on your system PATH. Use these safe fallback options:

Option A (Safe Direct Execution): python -m mtrini_cli
Option B (Fix PATH in PowerShell): [Environment]::SetEnvironmentVariable("PATH", $env:PATH + ";$env:APPDATA\Python\Python312\Scripts", "User")
